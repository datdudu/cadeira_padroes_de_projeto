Repository Pattern

Conceitos Fundamentais
•

Objetivo: Esconder completamente a lógica de persistência

•

API: Alto nível para consulta e manipulação de agregados

•

Perspectiva: "O que esse objeto de domínio pode fazer"

•

Abstração: Objetos do domínio como "coleções"

Repository vs DAO/Mapper
Diferenças Chave
• Repository: Métodos de negócio
• DAO: Operações CRUD diretas
• Repository: Trabalha com agregados
• DAO: Trabalha com tabelas
Arquitetura em Camadas
1

Repository → API de domínio

2

DAO → Operações de persistência

3

Mapper → Tradução objeto ↔ DB

Fluxo de Dados
Domínio → Repository → DataMapper → DAO → Banco
LivroRepository.java - Interface de Domínio
public interface LivroRepository {
    // Métodos orientados ao domínio
    void salvar(Livro livro);           // Inserir ou atualizar
    void remover(Livro livro);          // Remove objeto
    Optional<Livro> buscarPorId(Long id);
    List<Livro> buscarTodos();
    
    // Métodos específicos do domínio
    List<Livro> buscarDisponiveis();
    List<Livro> buscarPorAutor(String autor);
}
Repository Implementation - Lógica de Negócio
@Override
public void salvar(Livro livro) {
    // Lógica de negócio: inserir ou atualizar
    if (livro.getId() == null) {
        inserir(livro);
    } else {
        atualizar(livro);
    }
}

private void inserir(Livro livro) {
    // Aqui usa DAO/Mapper para persistência
    String sql = "INSERT INTO livros (titulo, autor, isbn, disponivel) VALUES (?, ?, ?, ?)";
    try (PreparedStatement stmt = connection.prepareStatement(sql)) {
        stmt.setString(1, livro.getTitulo());
        stmt.setString(2, livro.getAutor());
        stmt.setString(3, livro.getIsbn());
        stmt.setBoolean(4, livro.isDisponivel());
        stmt.executeUpdate();
    }
}
Uso via Service Layer
public class BibliotecaService {
    private final LivroRepository repository;

    public void emprestarLivro(Long livroId) {
        // Usa Repository como coleção de objetos
        Livro livro = repository.buscarPorId(livroId)
            .orElseThrow(() -> new IllegalArgumentException("Livro não encontrado"));
        
        // Aplica regra de negócio na entidade
        livro.emprestar();
        
        // Persiste mudanças via Repository
        repository.salvar(livro);
    }
}
