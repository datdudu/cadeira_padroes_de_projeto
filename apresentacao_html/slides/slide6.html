Repository: Agregados e Invariantes

Conceitos Fundamentais
•

Agregado: Objeto de domínio + todas suas dependências como unidade única

•

Invariante: Regras de negócio que sempre devem ser verdadeiras

•

Raiz do Agregado: Única forma de acessar entidades internas

Exemplo: Pedido como Agregado
• Raiz: Pedido
• Dependentes: ItemPedido
• Regra: ItemPedido só existe dentro de um Pedido
Invariantes do Pedido
1
Pedido deve ter pelo menos 1 item para ser confirmado
2
Valor total deve ser sempre a soma dos itens
3
Só pode adicionar itens se status for RASCUNHO
4
ItemPedido não pode ter quantidade ≤ 0
Pedido.java - Invariantes Protegidas
public void adicionarItem(String produtoId, int qtd, BigDecimal preco) {
    // INVARIANTE 3: Status deve ser RASCUNHO
    if (status != StatusPedido.RASCUNHO) {
        throw new IllegalStateException(
            "Não é possível adicionar itens a um pedido " + status);
    }
    
    // INVARIANTE 4: Quantidade deve ser > 0
    if (qtd <= 0) {
        throw new IllegalArgumentException(
            "Quantidade deve ser maior que zero");
    }
    
    ItemPedido item = new ItemPedido(produtoId, qtd, preco);
    itens.add(item);
    recalcularValorTotal(); // INVARIANTE 2
}
Repository - Gerencia Agregado Completo
public Optional<Pedido> buscarPorId(Long id) {
    // Recupera AGREGADO COMPLETO (pedido + itens)
    Pedido pedido = mapearPedido(buscarPedidoSql(id));
    List<ItemPedido> itens = buscarItensPedido(id);
    
    // Restaura agregado com factory method
    return Optional.of(Pedido.restaurarComItens(
        pedido.getId(), pedido.getClienteId(), 
        pedido.getStatus(), itens));
}
Service - Usa Agregado via Repository
public void adicionarItemAoPedido(Long pedidoId, String produtoId) {
    // Recupera agregado completo
    Pedido pedido = repository.buscarPorId(pedidoId)
        .orElseThrow(() -> new IllegalArgumentException("Pedido não encontrado"));
    
    // Modifica através da raiz (invariantes protegidas)
    pedido.adicionarItem(produtoId, 2, new BigDecimal("29.90"));
    
    // Salva agregado completo
    repository.salvar(pedido);
}
