DataMapper Pattern

Conceitos Fundamentais
•

Objetivo: Separar totalmente lógica de domínio da persistência

•

Função: Mapeamento entre objeto de domínio e objeto de persistência

•

Característica: Domínio não tem setters, apenas getters

•

Integração: Pode ser usado junto ao DAO

Diferenças do DAO
DAO vs DataMapper
• DAO: Mistura mapeamento com SQL
• DataMapper: Separa mapeamento em método específico
• DAO: Usa setters na entidade
• DataMapper: Usa construtor/factory methods
Vantagens
•

Domínio puro sem dependência de persistência

•

Facilita testes unitários

•

Reutilização de mapeamento

Livro.java - Domínio Puro
public class Livro {
    private final Long id;
    private final String titulo;
    private final String autor;
    private boolean disponivel;

    // Construtor (sem setters)
    public Livro(Long id, String titulo, String autor, boolean disponivel) {
        this.id = id;
        this.titulo = titulo;
        this.autor = autor;
        this.disponivel = disponivel;
    }

    // Métodos de negócio
    public void emprestar() {
        if (!disponivel) throw new IllegalStateException("Livro já emprestado");
        this.disponivel = false;
    }

    // Apenas getters
    public Long getId() { return id; }
    public String getTitulo() { return titulo; }
}
LivroMapper.java - Mapeamento Separado
public interface LivroMapper {
    void insert(Livro livro);
    void update(Livro livro);
    Optional<Livro> findById(Long id);
    List<Livro> findAll();
}

// Implementação com mapeamento separado
private Livro mapRow(ResultSet rs) throws SQLException {
    return new Livro(
        rs.getLong("id"),
        rs.getString("titulo"),
        rs.getString("autor"),
        rs.getBoolean("disponivel")
    );
}

@Override
public void update(Livro livro) {
    String sql = "UPDATE livros SET titulo=?, autor=?, disponivel=? WHERE id=?";
    try (PreparedStatement stmt = connection.prepareStatement(sql)) {
        // Usa getters (não setters)
        stmt.setString(1, livro.getTitulo());
        stmt.setString(2, livro.getAutor());
        stmt.setBoolean(3, livro.isDisponivel());
        stmt.setLong(4, livro.getId());
        stmt.executeUpdate();
    } catch (SQLException e) {
        throw new RuntimeException("Erro ao atualizar livro", e);
    }
