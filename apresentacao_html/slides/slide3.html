DAO - Data Access Object

Conceitos Fundamentais
•

Objetivo: Abstrair acesso à fonte de dados (persistência)

•

Flexibilidade: Implementação pode ser alterada sem modificar cliente

•

Limitação: Domínio NÃO é separado da lógica de acesso

•

Característica: Usa setters, modifica entidade diretamente

Características do DAO
Problemas Típicos
• Mapeamento inline: Misturado com SQL
• Setters na entidade: Quebra encapsulamento
• Domínio contaminado: Lógica de persistência presente
• Acoplamento: Objeto de domínio = objeto de persistência
Operações Típicas
•

Operações CRUD básicas (Create, Read, Update, Delete)

•

Consultas simples por campos específicos

•

Mapeamento direto tabela ↔ objeto

Problema: Mapeamento Inline + Setters
@Override
public List<Livro> buscarTodos() throws SQLException {
    String sql = "SELECT * FROM livros";
    List<Livro> livros = new ArrayList<>();

    try (Statement stmt = connection.createStatement()) {
        ResultSet rs = stmt.executeQuery(sql);
        while (rs.next()) {
            // ❌ Mapeamento inline (misturado com SQL)
            Livro livro = new Livro();
            livro.setId(rs.getLong("id"));
            livro.setTitulo(rs.getString("titulo"));
            livro.setAutor(rs.getString("autor"));
            livro.setIsbn(rs.getString("isbn"));
            livro.setDisponivel(rs.getBoolean("disponivel"));
            livros.add(livro);
        }
    }
    return livros;
}
Livro.java - Entidade com Setters
public class Livro {
    private Long id;
    private String titulo;
    private String autor;
    private String isbn;
    private boolean disponivel;

    // ❌ Setters expõem estado interno
    public void setId(Long id) { this.id = id; }
    public void setTitulo(String titulo) { this.titulo = titulo; }
    public void setAutor(String autor) { this.autor = autor; }
    public void setDisponivel(boolean disponivel) { 
        this.disponivel = disponivel; 
    }

    // ✅ Getters e métodos de negócio
    public Long getId() { return id; }
    public String getTitulo() { return titulo; }
    
    public void emprestar() {
        if (!disponivel) throw new IllegalStateException("Livro já emprestado");
        this.disponivel = false;
    }
}
